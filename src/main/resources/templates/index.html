Simulado Online
Sobre

Simulado Online é uma plataforma gratuita para treinar questões de concursos e provas.

De forma rápida e simples, o usuário escolhe as áreas de conhecimento, define a quantidade de questões e o tempo, e já começa o simulado — sem precisar criar conta ou fazer login.

As questões são sorteadas aleatoriamente do banco de dados a cada prova, garantindo variedade e imparcialidade. Ao final, o participante vê um relatório com acertos, erros, porcentagem e gabarito detalhado.

Tecnologias: Java + Spring Boot, PostgreSQL, Thymeleaf e JavaScript.

Objetivo: tornar o treino de simulados mais acessível e prático para qualquer candidato.
